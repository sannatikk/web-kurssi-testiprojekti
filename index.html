My Work & Experience

Feel free to browse my GitHub repositories for examples of schoolwork I've been doing! I'm currently learning about web development, cloud services, and Java.

My favorite project so far has been an ATM simulator, where I got to practise building a Qt app, using Node.js, building and querying relational databases, and using RESTful APIs. Fun extra credit challenges included hosting the database and backend remotely, using custom style sheets, and making the app ensuring cross-platform compatibility.

Another fun project was the Arduino-based game "Speden Spelit", which features LEDs, buttons, and a 7-segment display. Pressing the start button initiates LEDs blinking in random order at a steadily increasing pace. Players must press the gameplay buttons in the correct sequence to accumulate points, and the game ends if a mistake is made; the 7-segment display shows the score and signals if a new high score is achieved.
